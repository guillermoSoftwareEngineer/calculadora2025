# Calculadora Eléctrica | Angular

Aplicación web desarrollada con Angular para realizar cálculos eléctricos fundamentales en sistemas monofásicos y trifásicos. Permite obtener valores de potencia, corriente, voltaje y resistencia a partir de variables ingresadas por el usuario. El objetivo principal del proyecto es ofrecer una herramienta intuitiva, precisa y rápida para estudiantes, técnicos e ingenieros que necesitan validar datos eléctricos sin recurrir a software especializado.

## Características principales

1. Cálculo de potencia, voltaje, corriente y resistencia.
2. Soporte para sistemas monofásicos y trifásicos.
3. Selección del tipo de conexión.
4. Validación de datos en tiempo real mediante formularios reactivos.
5. Componentes modulares desarrollados con Angular.
6. Interfaz responsiva compatible con dispositivos móviles y de escritorio.
7. Lógica matemática implementada en TypeScript mediante funciones reutilizables.

## Tecnologías utilizadas

- Angular
- TypeScript
- HTML
- CSS
- JavaScript

## Arquitectura y organización

El proyecto está organizado bajo principios de modularización para mantener claridad, escalabilidad y separación de responsabilidades. La lógica de cálculo se encuentra implementada en servicios específicos. Los componentes manejan la presentación y la interacción con el usuario. Los formularios se gestionan mediante Reactive Forms para asegurar validación y control total de los datos.

## Retos abordados y aprendizajes

1. Implementación de cálculos eléctricos dinámicos con TypeScript.
2. Gestión de estado y validación mediante formularios reactivos.
3. Diseño de una interfaz clara para usuarios no técnicos.
4. Modularización del proyecto para mejorar la mantenibilidad.
5. Separación de lógica y presentación siguiendo buenas prácticas de Angular.

## Instalación y ejecución del proyecto

Clonar el repositorio:

```bash
git clone https://github.com/guillermoSoftwareEngineer/calculadora2025
```

Instalar dependencias:

```bash
npm install
```

Ejecutar el servidor de desarrollo:

```bash
ng serve
```

Abrir en el navegador:

http://localhost:4200/
